== Description ==

VC Mega Footer lets you use WPBakery Pagebuilder (formerly known as Visual Composer) to create your site footer.

Use multiple Footer Blocks that can appear on all pages, or only selected ones.

Comes with 12 pre-built footer layouts to get you going.

== Installation ==

1. Upload to the `/wp-content/plugins/` directory (this plugin is an add on for WPBakery Pagebuilder)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click the new 'Mega Footer' menu element in your wordpress admin
4. You are ready to make your footer mega using WPBakery Pagebuilder
5. Add Footer Blocks, these can show on all pages, or just some pages
